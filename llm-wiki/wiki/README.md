# 위키

이 디렉터리는 LLM이 원천 자료를 바탕으로 생성하고 유지하는 지식 레이어다.

주요 파일:

- `index.md`: 위키 페이지의 콘텐츠 중심 카탈로그.
- `overview.md`: 위키의 현재 상태에 대한 최상위 종합.
- `log.md`: ingest, query, lint 같은 위키 작업 기록.

주요 디렉터리:

- `sources/`: 원천 자료별 요약.
- `entities/`: 사람, 조직, 제품, 장소 등 명명된 대상.
- `concepts/`: 아이디어, 이론, 용어, 패턴, 프레임워크.
- `syntheses/`: 여러 원천 자료를 엮은 분석과 종합.
- `questions/`: 오래 보존할 질문과 답변.
- `_templates/`: 위키 페이지 유형별 작성 템플릿.

원천 파일은 `raw/`에 두고, 요약과 해석은 이 디렉터리에 작성한다.
